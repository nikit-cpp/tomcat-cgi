I run it in IntelliJ IDEA

1) Check that your Linux can execute
web/WEB-INF/cgi/hi-perl.cgi
if no, type
chmod +x web/WEB-INF/cgi/hi-perl.cgi

2) Open in browser
http://localhost:8080/cgi-bin/hi-perl.cgi
